# Credit-card-customer-segmentation

## Project Overview

This project explores customer segmentation using unsupervised machine learning techniques. The goal is to identify groups of credit card customers with similar financial behavior, enabling businesses to better understand their customer base and support data-driven marketing strategies.

The analysis covers the complete data science workflow, including data preprocessing, exploratory data analysis, dimensionality reduction, clustering, and interpretation of the resulting customer segments.

## Objectives

- Clean and preprocess real-world customer data
- Perform exploratory data analysis (EDA)
- Detect and handle missing values and outliers
- Reduce dimensionality using Principal Component Analysis (PCA)
- Apply multiple clustering algorithms
- Compare clustering results
- Interpret customer segments from a business perspective

## Dataset

The dataset contains credit card customer information, including variables related to:

- Balance
- Purchases
- Cash advances
- Credit limits
- Payments
- Purchase frequency
- Other customer spending characteristics

## Key Results
- Identified distinct groups of customers based on purchasing behavior.
- Reduced dataset dimensionality using PCA while preserving the most important information.
- Compared K-Means and Gaussian Mixture Models for customer segmentation.
- Interpreted the discovered customer groups to provide business-relevant insights.
